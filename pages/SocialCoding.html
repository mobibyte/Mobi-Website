Social Coding

Social Coding is held almost every Friday during the Fall and Spring semesters (Here's our calendar). It's a time where students that know a little bit about a framework, language, or platform can team up and create a fun and exciting project.

What We Use

These are just a few of many technologies we use during social coding. Feel free to stop by and learn about some of your favorites.

Mobile

We like to develop applications using native as well as hybrid technologies for mobile connected devices.

Web Development

HTML, CSS, and Javascript has changed the web, so we are extremely adept to using these technologies.

Server Side

We experiment with javascript based frameworks for server-side development and databases.

Planning

One important step in the develoment lifecylce is to plan. During social coding we plan to build and anyone is welcome to join! Come join us to help plan our next project!

Build

Build real world projects in a group setting and learn technologies that are used in the industry! Learn how real software teams work collaboratively with technologies like Git.

Launch

After the project is completed we'll deploy it so you can show off your skills to potential employers! Having a real world working application can help you stand out!

Join the Team

Join Social Coding! Come in and discuss cool things your working on. You don't have to be a pro to join social coding. We'll give you some resources to help! Social coders team up and go to hackathons to win prizes and network with companies.
